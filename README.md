# 💰 Expense Tracker - Personal Finance App

A simple console-based Expense Tracker built using Python. This project allows users to record daily expenses, view all transactions, calculate total spending, and analyze spending by category.

## 🚀 Features

- ✅ Add new expenses
- ✅ View all recorded expenses
- ✅ Calculate total spending
- ✅ View spending by category
- ✅ Interactive menu-driven interface

## 🛠️ Built With

- Python 3
- Lists
- Dictionaries
- Loops
- Conditional Statements

## 📋 Menu

```
1. Add Expense
2. View All Expenses
3. View Total Spending
4. View Spending by Category
5. Exit
```
